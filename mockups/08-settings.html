AI / LLM

Stow uses a local OpenAI-compatible inference server for NL entry and bank import parsing. No data leaves your machine.

Ollama default is http://localhost:11434/v1 · oMLX default is http://localhost:10240/v1

The model must support function calling (tool use). Qwen3, Llama 3.1+, Mistral v0.3+ all work.
